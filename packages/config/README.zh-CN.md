# @visual-guard/config

> Visual Guard config package

## 安装

```bash
npm install @visual-guard/config
# 或
pnpm add @visual-guard/config
```

## 用法

```ts
import {helloConfig} from '@visual-guard/config';

console.log(helloConfig('读者'));
```
