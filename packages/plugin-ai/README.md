# @visual-guard/plugin-ai

> Visual Guard plugin-ai package

## Install

```bash
npm install @visual-guard/plugin-ai
# or
pnpm add @visual-guard/plugin-ai
```

## Usage

```ts
import {helloPluginAi} from '@visual-guard/plugin-ai';

console.log(helloPluginAi('reader'));
```
