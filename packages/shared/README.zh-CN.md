# @visual-guard/shared

> Visual Guard shared package

## 安装

```bash
npm install @visual-guard/shared
# 或
pnpm add @visual-guard/shared
```

## 用法

```ts
import {helloShared} from '@visual-guard/shared';

console.log(helloShared('读者'));
```
