# @visual-guard/reporters

> Visual Guard reporters package

## Install

```bash
npm install @visual-guard/reporters
# or
pnpm add @visual-guard/reporters
```

## Usage

```ts
import {helloReporters} from '@visual-guard/reporters';

console.log(helloReporters('reader'));
```
