# @visual-guard/plugin-archive

> Visual Guard plugin-archive package

## Install

```bash
npm install @visual-guard/plugin-archive
# or
pnpm add @visual-guard/plugin-archive
```

## Usage

```ts
import {helloPluginArchive} from '@visual-guard/plugin-archive';

console.log(helloPluginArchive('reader'));
```
