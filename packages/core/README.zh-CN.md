# @visual-guard/core

> Visual Guard core package

## 安装

```bash
npm install @visual-guard/core
# 或
pnpm add @visual-guard/core
```

## 用法

```ts
import {helloCore} from '@visual-guard/core';

console.log(helloCore('读者'));
```
