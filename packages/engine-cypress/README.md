# @visual-guard/engine-cypress

> Visual Guard engine-cypress package

## Install

```bash
npm install @visual-guard/engine-cypress
# or
pnpm add @visual-guard/engine-cypress
```

## Usage

```ts
import {helloEngineCypress} from '@visual-guard/engine-cypress';

console.log(helloEngineCypress('reader'));
```
