# @visual-guard/engine-puppeteer

> Visual Guard engine-puppeteer package

## 安装

```bash
npm install @visual-guard/engine-puppeteer
# 或
pnpm add @visual-guard/engine-puppeteer
```

## 用法

```ts
import {helloEnginePuppeteer} from '@visual-guard/engine-puppeteer';

console.log(helloEnginePuppeteer('读者'));
```
