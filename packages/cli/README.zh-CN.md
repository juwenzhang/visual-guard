# @visual-guard/cli

> Visual Guard cli package

## 安装

```bash
npm install @visual-guard/cli
# 或
pnpm add @visual-guard/cli
```

## 用法

```ts
import {helloCli} from '@visual-guard/cli';

console.log(helloCli('读者'));
```
